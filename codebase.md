# Codebase Index

> Shopify theme at `src/`. Update this file whenever a file is created or deleted in `src/`.

## Config & Build (root of `src/`)

| File | Description |
|---|---|
| `src/package.json` | npm config — scripts: `dev` (Tailwind watch + shopify dev), `build:css`, `watch:css` |
| `src/tailwind.config.js` | Tailwind v3 config — content globs scan all `.liquid` + `.js`; theme extends CSS tokens |
| `src/styles/main.css` | Tailwind input source — declares `@layer base, components, utilities` |
| `src/.gitignore` | Ignores `.shopify/`, `node_modules/`, `assets/main.css` (compiled output) |
| `src/.prettierrc.json` | Prettier formatting rules |
| `src/.theme-check.yml` | Shopify Theme Check linter config |
| `src/translation.yml` | Crowdin translation config |

## `src/locales/` — Translations

Contains `*.json` files for each locale (e.g. `en.default.json`, `de.json`, `fr.json`, etc.) and `*.schema.json` files for translatable settings labels. All UI strings must use `{{ 'key' | t }}` — never hardcode copy.
